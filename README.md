
C++ library for manipulating PGM images.

Implemented operations:
## Brightness and Contrast Adjustments
## Gamma Correction
## Convolutional Kernels
* Mean Blur
* Gaussian Blur
* Vertical Sobel
* Horizontal Sobel

## Drawing module
Allows for drawing primitives on the image.
Implemented primitives include:
* Rectangle
* Circle
* Line
